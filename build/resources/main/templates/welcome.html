Rules of Hangman:

  1. You are going to guess a word, which is presented by a row of dashes and each dash represents each letter of the word.
  2. You will guess letters which occur in the word.
  3. If the letter does occur in the word, it will present the position(s) of the letter occuring in the word.
  4. If the letter does not occur in the word, it increments the number of incorrect guesses.
  5. You have a maximum of 7 incorrect guesses.
